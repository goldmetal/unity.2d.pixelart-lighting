# 픽셀아트를 위한 2D 라이팅
![Unity Version](https://img.shields.io/badge/Unity-6000.5.2f1-202020?&logo=unity&logoColor=white)![Assets](https://img.shields.io/badge/Pixel_Art-FFD700?&logoColor=white)![Visual Scripting](https://img.shields.io/badge/Visual_Scripting-FFBF0F?&logo=visual-studio-code&logoColor=white)  
![Banner](url)  
유니티 URP 2D의 주요 기능인 Light 2D를 적극 활용하여
스프라이트만으로는 표현할 수 없는 멋진 장면을 연출하는 방법에 대해 익힐 수 있는 예제 프로젝트입니다.  
  
장면은 2개로 구분되어 있으며 학습을 위해서라면 Basic 장면ㅇ로 시작하시는 것을 추천드립니다.  
**- Basic**: 아무런 Light 2D가 없는 장면  
**- Completed**: 모든 Light 2D가 배치되어 완벽한 분위기를 갖춘 장면  
  
커리큘럼은 아래 링크를 통해 Unity Learn 플랫폼에서 학습하실 수 있습니다.  
모든 학습 내용을 정복하고 최종 결과물을 **Unity Play**에 제출하시면 Unity Learn 생태계에 많은 도움이 됩니다.  
**[Unity Learn](https://learn.unity.com/course/2d-lighting-for-pixel-art-kr)**  
  
해당 프로젝트에 등장하는 캐릭터는 Dragon Crusher 에셋의 용사 3인방 중 하나인 마법사 **Griorgia** 입니다.  
**[Dragon Crusher](https://assetstore.unity.com/packages/essentials/tutorial-projects/dragon-crashers-urp-2d-sample-project-190721)**
